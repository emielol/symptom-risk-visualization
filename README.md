This project is an end-to-end disease diagnosis and risk visualization system that allows users to:

- Select symptoms through an interactive UI

- Receive ranked disease predictions from a trained ML model

- Explore relative confidence levels via dynamic visualizations

- Inspect how individual symptoms contribute to predictions

The goal of this project is not only prediction, but interpretability and communication of model behavior.
